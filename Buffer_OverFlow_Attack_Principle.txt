This is the Buffer OverFlow attack principle.
The key point is to find the address of EIP/ESP and R14/R13.


X86-64 Arch to detect EIP and ESP 

-----------------------------------------------------
			|			|			|	 |			|
			|			|			|	 |			|
	BUF     |   EBP     |  EIP      |PAD |   ESP	|
			|			|			|	 |			|
-----------------------------------------------------

ARM Arch to detect R14 and R13

-----------------------------------------------------
			|			|			|	 |			|
			|			|			|	 |			|
	BUF     |   R11     |  R14      |PAD |   R13	|
			|			|			|	 |			|
-----------------------------------------------------
